О чём пойдёт речь

Два постулата:

  1. Если что-то можно сделать через CSS – лучше делать это через CSS.
  2. Знание JavaScript не может заменить знание CSS.

Чек-лист

Автор пишет если слова незнакомы, то надо стопнуть, однако сам он разбирал ряд этой свойств в учебнике.

  • Блочная модель, включая:
    • margin, padding, border, overflow
    • а также height/width и min-height/min-width.
  • Текст:
    • font
    • line-height.
  • Различные курсоры cursor.
  • Позиционирование:
    • position, float, clear, display, visibility
    • Центрирование при помощи CSS
    • Перекрытие z-index и прозрачность opacity
  • Селекторы:
    • Приоритет селекторов
    • Селекторы #id, .class, a > b
  • Сброс браузерных стилей, reset.css
