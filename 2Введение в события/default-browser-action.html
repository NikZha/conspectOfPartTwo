Действия браузера по умолчанию

Ряд событий влекут за собой автоматические действия браузера по умолчанию. Зачастую эти действия можно отменить при помощи javascript.

Отмена действия браузера

Два способа отмены:

  • onclick="return false" причём подобный возврат в других случаях игнорируется.
  • onclick="event.preventDefault()"
