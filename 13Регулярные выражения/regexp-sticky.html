Поиск на заданной позиции, флаг "y"

Флаг y позволяет произвести поиск на определённой позиции в исходной строке. То есть как следует из примера, происходит поиск регекспа с заданного индекса:

 
                
                let str = 'let varName = "value"';

                let regexp = /\w+/y;
                for (let index = 0; index < str.length; index++) {
                    regexp.lastIndex = index;
                    alert(regexp.exec(str) + "Индекс равен: " + index)
                }

Автор пишет, что такое бывает полезно для производительности.
