Рассмотрение соотвествующих событий.

Событие: change

Срабатывает по окончатнию измнения документа, для полей ввода типа <input> означает потерю фокуса:


<input type="text" onchange="alert(this.value)">
<input type="button" value="Button">

Для других элемментов типа select, input type=checkbox, radio событие срабатывает сразу после изменения значения.
