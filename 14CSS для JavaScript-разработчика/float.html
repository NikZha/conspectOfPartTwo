Свойство float

Важное свойство которое позволяет распределять блоки по странице не прибегая к услугам таблиц.

Как работает float

Синтаксис:


float: left | right | none | inherit;

Как работает свойство:

  1. Элемент позиционируется, как обычно, а потом вынимается из потока и сдвигается или влево (для left) или вправо до границы или до другого элемента с float.
  2. Если не хватает пространства по горизонтали, то элемент переносится на следующую строку.
  3. Блочные элементы без float ведут себя так, будто элемнета с float нет, так как он убран из потока.
  4. inline-элементы (строки и проч) обтекают float элемент.

А так же:

  1. Элемент с float так же получается свойство: display:block в не зависимости от других свойств. Таким образом на него будет работать свойства width/height.
  2. Ширина элемента определяется его содержимым.
  3. Вертикальные отступы margin элементов с float не сливаются с отступами соседей, в отличие от обычных блочных элементов.

Добавление свойства clear

Для решения проблемы используется свойство clear:


clear: left | right | both;

Применяемое к элементам:


h2 {
  clear: left;
}

Таким образом получается:

Но получается неидеально - второй заголовок сильно прилеплен к верхней картинке. Чтобы было лучше надо добавить дополнительный разделяющих элемент: 


<div style="clear:both"></div>

Который решит эту проблему:

  • Свойство clear гарантировало, что <div style="clear:both"> будет под картинкой с float.
  • Заголовок <h2>Сова</h2> идёт после этого <div>. Так что его отступ учитывается.

Более похоже на кастыль, чем на решение.

Заполнение блока-родителя

Есть ещё одна засада, как видно из примера:

Элемент float выпадает из границ родительского блока:


    <style>
        .hero {
            background: #D2B48C;
            border: 1px solid red;
        }
    </style> 
    <div class="hero">
        <h2>Ленин, Владимир Ильич</h2>
        <div class="left border">
            <br>Портрет <br>Владимира Ильича
            <br>Ленина
        </div>
        <p>Влади́мир Ильи́ч Ле́нин (фамилия при рождении — Улья́нов; 10 (22) апреля 1870, ..... </p>
    </div>

Для решения этой проблемы используются следующие техники:

Поставить родителю float


.hero {
  background: #D2B48C;
  border: 1px solid red;
  float: left;
}

Добавить в родителя элемент с clear

Добавим элемент div style="clear:both" в самый конец контейнера .hero. Минусом этой техники является лишний элемент.

Универсальный класс clearfix

Чтобы не добавлять в HTML-код лишний элемент, можно задать его через :after.


.clearfix:after {
  content: "."; /* добавить содержимое: "." */
  display: block;  /* сделать блоком, т.к. inline не может иметь clear */
  clear: both;  /* с обеих сторон clear */
  visibility: hidden; /* сделать невидимым, зачем нам точка внизу? */
  height: 0;  /* сделать высоту 0, чтобы не занимал место */
}

Добавив этот класс к родителю, получим тот же результат.

overflow:auto/hidden

Если добавить родителю overflow: hidden или overflow: auto, то всё станет хорошо.


.hero {
  overflow: auto;
}

Вёрстка в несколько колонок

Автор рассматривает два метода верстки:

float:left + float:right

HTML-код примерно такой:


<div>Шапка</div>
<div class="column-left">Левая колонка</div>
<div class="column-right">Правая колонка</div>
<div class="footer">Низ</div>

CSS такой:


.column-left {
  float: left;
  width: 30%;
}

.column-right {
  float: left;
  width: 70%;
}

.footer {
  clear: both;
}

float + margin

Того же самого можно добиться оставив поле для правой колонки через margin:


.column-left {
  float: left;
  width: 30%;
}

.column-right {
  margin-left: 30%;
}

.footer {
  clear: both;
}
