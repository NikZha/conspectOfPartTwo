Якоря: начало строки ^ и конец $

Как следует из названия ^ означает начало строки, а $ конец строки, например:


let str1 = "Mary had a little lamb";
alert( /^Mary/.test(str1) ); // true
alert( /lamb$/.test(str1) ); // true

В конкретно этих случаях мы могли бы использовать и методы строк startsWith/endsWith. Регулярные выражения следует применять, когда нужна проверка сложнее.

Проверка на полное совпадение

Оба якоря вместе ^...$ используют чтобы проверить строку на полное совпадение с паттерном: 


let goodInput = "12:34";
let badInput = "12:345";

let regexp = /^\d\d:\d\d$/;
alert( regexp.test(goodInput) ); // true
alert( regexp.test(badInput) ); // false

У самих якорей нулевая длинна.

Задача


                let str = ''
                let regexp = /^$/;
                alert(regexp.test(str))

Итого паттерн /^$/соотвествует пустой строке.
