Граница слова: \b

Проверка на границы слова:


alert( "Hello, Java!".match(/\bHello\b/) ); // Hello
alert( "Hello, Java!".match(/\bJava\b/) );  // Java
alert( "Hello, Java!".match(/\bHell\b/) );  // null (нет совпадения)
alert( "Hello, Java!".match(/\bJava!\b/) ); // null (нет совпадения)
alert(' Тест '.match(/\bТест\b/))    //не ищет не латиницу

Как видно из этого примера:

  1. Границами отмечают как начало, так и конец слова.
  2. Если паттерн найден, то возвращается слово найденное паттерном. Если паттерн не найден возвращается null.
  3. Не работает на символы, но работает на цифры.
  4. Не латинские знаки оно не ищет.

Как было написано, можно искать цифры:


alert( "1 23 456 78".match(/\b\d\d\b/g) ); // 23,78
alert( "12,34,56".match(/\b\d\d\b/g) ); // 12,34,56
