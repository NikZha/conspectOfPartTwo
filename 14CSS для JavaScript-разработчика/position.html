Свойство position

Свойство position позволяет сдвигать элемент со своего обычного места.

position: static

Позиционирование по умолчанию всех элементов. Отдельно указывается редко:


position: static;

В основном для переопределния других значений position.

position: relative

Относительное позиционирование position: relative в css. Так же с этим свойством указывают куда сдвигают: left/right/top/bottom, например: 


<style>
  #exmpl1 {
    position: relative;
    top: 10px;
  }
</style>

<div style="background: #fee; width: 500px">
    Заголовок сдвинут на 10px вниз.

    <h2 style="background: #aef; margin: 0;" id="exmpl1">Заголовок</h2>

    <div>А тут - всякий разный текст... <br/>
         ... В две строки!</div>
</div>
Заголовок сдвинут на 10px вниз.

Заголовок

А тут - всякий разный текст...
... В две строки!

Координаты:

  • top – сдвиг от «обычной» верхней границы
  • bottom – сдвиг от нижней границы
  • left – сдвиг слева
  • right – сдвиг справа

Можно сдвигать с минусом, например left: 10% сдвинет элемент на 10% его ширины вправо, а left: -10% – влево.
