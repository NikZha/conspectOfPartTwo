FormData


let formData = new FormData([form]);            

Это конструктор создающий соотвествующий типа даных, если передаётся в этот конструктор html-элемент form, то прочитываются её поля.


При обмене данными с сетью указывается тип данных multipart/form-data как и в случае обычной форме в запросе  POST.

Отправка простой формы

Создание простой формы:


        <form id="formElem">
            <input type="text" name="name" value="John">
            <input type="text" name="surname" value="Smith">
            <input type="submit">
        </form>

        <script>
            formElem.onsubmit = async (e) => {
                e.preventDefault();

                let response = await fetch('/article/formdata/post/user', {
                    method: 'POST',
                    body: new FormData(formElem)
                });

                let result = await response.json();

                alert(result.message);
            };
        </script>

Пример у меня не работает, потому что нет серверной части которая принимает эту форму и отвечает на запрос.

Но в целом понятно, что действие по умолчанию отменяется, а сама форма создаётся в виде объекта FromData и отправляется при помощи fetch, а далее читается ответ сервера.
