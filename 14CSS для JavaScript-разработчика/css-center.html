Центрирование горизонтальное и вертикальное

В CSS есть всего несколько техник центрирования элементов. Если их знать, то большинство задач решаются просто.

Горизонтальное
text-align

Для инлайновых элементов используется text-align: center, например:


<style>
  .outer {
    text-align: center;
    border: 1px solid blue;
  }
    .inner {
    width: 100px;
    border: 1px solid red;
  }
</style>

<div class="outer">Текстно для блочных элементов такое не сработает:
                <div class="inner">Текст</div>
                </div>
Текст, но для блочных элементов такое не сработает:
Текст

Для блочных элементов используют:

margin: auto

Блок по горизонтали центрируется margin: auto:


<style>
  .outer {
    border: 1px solid blue;
  }
  .inner {
    width: 100px;
    border: 1px solid red;
    margin: auto;
  }
</style>

<div class="outer">
  <div class="inner">Текст</div>
</div>
Текст

Значение margin: auto нужно проставлять явно. Так же приспользовании свойств margin-left:auto/margin-right:auto будет использовано все свободное место с соотвествующего края.

Вертикальное

Вертикальное же изначально не было предусмотрено в спецификации CSS и по сей день вызывает ряд проблем. Есть три основных решения.

position:absolute + margin


<style>
  .outer {
    position: relative;
    height: 5em;
    border: 1px solid blue;
  }

  .inner {
    position: absolute;
    top: 50%;
    border: 1px solid red;
  }
</style>

<div class="outer">
  <div class="inner">Текст</div>
Текст

Получается не по цетру поэтому сдвигаем вверх на половину штрифта:


<style>
  .outer {
    position: relative;
    height: 5em;
    border: 1px solid blue;
  }

  .inner {
    position: absolute;
    top: 50%;
    margin-top: -0.625em;
    border: 1px solid red;
  }
</style>

<div class="outer">
  <div class="inner">Текст</div>
</div>
Текст

Можно аналогично центрировать и по горизонтали, если известен горизонтальный размер, при помощи left:50% и отрицательного margin-left.

Одна строка: line-height

Вертикально отцентрировать одну строку в элементе с известной высотой height можно, указав эту высоту в свойстве line-height:


<style>
  .outer {
    height: 5em;
    line-height: 5em;
    border: 1px solid blue;
  }
</style>

<div class="outer">
  <span style="border:1px solid red">Текст</span>
</div>
Текст

Это работает, но лишь до тех пор, пока строка одна, а если содержимое вдруг переносится на другую строку, то начинает выглядеть довольно уродливо.
