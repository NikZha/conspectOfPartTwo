Центрирование горизонтальное и вертикальное

В CSS есть всего несколько техник центрирования элементов. Если их знать, то большинство задач решаются просто.

Горизонтальное
text-align

Для инлайновых элементов используется text-align: center, например:


<style>
  .outer {
    text-align: center;
    border: 1px solid blue;
  }
    .inner {
    width: 100px;
    border: 1px solid red;
  }
</style>

<div class="outer">Текстно для блочных элементов такое не сработает:
                <div class="inner">Текст</div>
                </div>
Текст, но для блочных элементов такое не сработает:
Текст

Для блочных элементов используют:

margin: auto

Блок по горизонтали центрируется margin: auto:


<style>
  .outer {
    border: 1px solid blue;
  }
  .inner {
    width: 100px;
    border: 1px solid red;
    margin: auto;
  }
</style>

<div class="outer">
  <div class="inner">Текст</div>
</div>
Текст

Значение margin: auto нужно проставлять явно. Так же приспользовании свойств margin-left:auto/margin-right:auto будет использовано все свободное место с соотвествующего края.
